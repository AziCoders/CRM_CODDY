# Быстрый старт

## 1. Установка зависимостей

```bash
pip install aiogram==3.13.1 python-dotenv==1.0.0
```

## 2. Настройка .env

Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=ваш_токен_бота
OWNER_ID=ваш_telegram_id
NOTION_API_KEY=ваш_notion_api_key
```

## 3. Запуск

```bash
python bot/main.py
```

## 4. Тестирование

1. Отправьте `/start` боту с нового аккаунта
2. Введите ФИО
3. Владелец получит уведомление с кнопками выбора роли
4. Выберите роль для пользователя
5. Пользователь получит уведомление и доступ к меню

## Структура roles.json

После регистрации файл `roles.json` будет содержать:

```json
{
  "123456789": {
    "fio": "Иванов Иван Иванович",
    "username": "ivanov",
    "role": "teacher",
    "city": "Назрань"
  }
}
```
